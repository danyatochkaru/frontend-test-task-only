# Тестовое задание Only

_Для запуска проекта требуется Node.js 18 версии и выше_

Установка зависимостей

```
npm install
```

Запуск проекта

```
npm run dev
```

## О проекте

* Проект выполненен на языке TypeScript с использованием фреймворка React.
* Сборка проекта осуществляется с помощью Webpack.
* Для стилизации использован SCSS пре-процессор.
* Слайдер реализован с помощью библиотеки Swiper.
* Анимации реализованы с использованием библиотеки gsap.
* Для управления состоянием используется Redux Toolkit.
